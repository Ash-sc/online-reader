# 在线小说阅读
在线小说阅读器，前端使用React框架，后端使用Node.js(爬取http://www.snwx8.com/ 的小说内容)。    
示例地址：http://ashshen.cc/online-reader/
    
### 运行方式:
    
````
    $ npm installl
    $ npm run build
    $ node ./app.js
````
浏览器输入：http://localhost:4396
    
### 本地调试：
    
````
    $ npm install
    $ node ./webpack-dev-server.js
    $ node ./app.js
````
修改代码后实时更新
    
浏览器输入：http://localhost:8999
